\n`tor`的全称是[the onion router](https://torproject.org/),是用来保护互联网隐私的一个开源软件。\n\n\n这篇文章是在debian上安装使用tor代理的过程。\n\n\n## 安装\n\n\n在 `debian` 上安装 `tor` 只需要执行一条命令\n\n\n```\n`sudo apt-get install tor`\n```\n\n## 配置\n\n\n编辑torrc文件\n\n\n```\n`sudo vi /etc/torrc`\n```\n\n找到这行，去掉前面#号\n\n\n```\n`#ControlPort 9051`\n```\n\n找到这行，去掉前面#号，再把 `1` 改成 `0`\n\n\n```\n`#CookieAuthentication 1`\n```\n\n重启 `tor`\n\n\n```\n`sudo /etc/init.d/tor restart`\n```\n\n下面是跨机共享代理流量给其他设备，去掉前面#号即可开启\n\n\n```\n`#SocksPort 0.0.0.0:9050`\n```\n\n卸载tor\n\n\n```\n`sudo apt remove tor && sudo apt purge tor && sudo apt autoremove`\n```\n\n\n\n
