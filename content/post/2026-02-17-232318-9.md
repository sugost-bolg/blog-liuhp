\n\n\n\n---\n\n\ntitle: OpenClaw 一键备份还原系统\n\n\nslug: openclaw-backup-system\n\n\ncategories: 技术\n\n\ntags: OpenClaw, Git, 备份\n\n\n---\n\n\n\n\n\n# OpenClaw 一键备份还原系统\n\n\n\n\n\n为 OpenClaw 设计的一套极简备份方案，基于 Git 单仓标签管理，支持文件变更自动备份和一键重装恢复。\n\n\n\n\n\n## 设计原则\n\n\n\n\n\n- 单仓多分支：一个 Git 仓库，用标签管理版本\n\n\n- 自动轮转：保留最近 3 次 + 当前，老的自动清理\n\n\n- 零依赖自举：重装后一行命令就能开始恢复\n\n\n- 双轨推送：SSH 用于日常备份，Git Daemon 用于紧急恢复\n\n\n\n\n\n## 备份策略：标签轮转\n\n\n\n\n\n| 标签名 | 含义 | 生命周期 |\n\n\n|--------|------|----------|\n\n\n| `current` | 当前工作副本 | 持续更新 |\n\n\n| `backup-1` | 最新备份 | 保留 |\n\n\n| `backup-2` | 上次备份 | 保留 |\n\n\n| `backup-3` | 上上次备份 | 保留 |\n\n\n\n\n\n## 命令\n\n\n\n\n\n```bash\n\n\nbackup now          # 立即备份\n\n\nbackup list         # 列出可用备份\n\n\nbackup restore 2    # 恢复到指定版本\n\n\nbackup auto         # 启动自动监听\n\n\nbackup stop         # 停止自动监听\n\n\n```\n\n\n\n\n\n重装恢复\n\n\n\n\n\n```bash\n\n\ncurl git://10.10.10.230/openclaw-back up.git/restore.sh | bash\n\n\n```\n\n\n\n\n\n开机自启动\n\n\n\n\n\n通过 macOS LaunchAgent 实现，监听文件变更，5秒后自动备份。\n\n\n\n\n\n完整文章：blog-posts/2026-02-15-open claw-backup-system.md\n\n
