\n查看防火墙状态:\n\n
\n```\n`firewall-cmd --state
`\n```\n
\n安装防火墙:\n\n
\n```\n`yum -y install firewalld
`\n```\n
\n启动:\n\n
\n```\n`systemctl start firewalld.service
`\n```\n
\n重启:\n\n
\n```\n`systemctl restart firewalld.service
`\n```\n
\n开机启动:\n\n
\n```\n`systemctl enable firewalld.service
`\n```\n
\n停止:\n\n
\n```\n`systemctl stop firewalld.service
`\n```\n
\n禁止开机启动:\n\n
\n```\n`systemctl disable firewalld.service
`\n```\n
\n查看状态:\n\n
\n```\n`systemctl status firewalld.service
`\n```\n
\n开启常用端口：\n\n
\n```\n`firewall-cmd --zone=public --add-port=22/tcp --permanent
firewall-cmd --zone=public --add-port=9443/tcp --permanent
firewall-cmd --zone=public --add-port=8443/tcp --permanent
firewall-cmd --zone=public --add-port=7443/tcp --permanent
`\n```\n
\n放行端口范围（8000-9000）：\n\n
\n```\n`firewall-cmd --zone=public --add-port=8000-9000/tcp --permanent
`\n```\n
\n查看已经放行的端口：\n\n
\n```\n`firewall-cmd --zone=public --list-ports
`\n```\n
\n删除某个已经放行的端口（6022）：\n\n
\n```\n`firewall-cmd --zone=public --remove-port=6022/tcp --permanent
`\n```\n
\n阻止某个IP（123.57.22.204）连接：\n\n
\n```\n`firewall-cmd --permanent --add-rich-rule="rule family=ipv4 source address=123.57.22.204 reject"
`\n```\n
