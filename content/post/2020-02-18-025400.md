\nMTProxy TLS一键安装绿色极简脚本\n\n
\n这个脚本相对于之前其他人发的，额外集成了mtproxy的新特性，根据指定的域名自动进行伪装。\n\n
\n减少被运营商检测的风险，更优于普通的代理或者随机字串数据包的形式。\n\n
\n需要注意的是，这里的TLS是假的（Fake TLS）。这会让你的IP访问起来更像是一个正常的网站。\n\n
\n所以，在这里推荐你使用默认端口：443，如果端口被占用或者NAT内网主机，可以设置其他端口。\n\n
\n除过443以外，还推荐你使用如下端口，都是常见的https端口，也可以进行一定程度的伪装：\n\n
\n如：443、2053、2083、2087、2096、8443\n\n
\n安装方式
执行如下代码进行安装\n\n
\n```\n`mkdir /home/mtproxy && cd /home/mtproxy
curl -s -o mtproxy.sh https://raw.githubusercontent.com/ellermister/mtproxy/master/mtproxy.sh && chmod +x mtproxy.sh && bash mtproxy.sh
mtproxy.sh
`\n```\n
\n使用方式
运行服务\n\n
\n```\n`bash mtproxy.sh start
`\n```\n
\n调试运行\n\n
\n```\n`bash mtproxy.sh debug
`\n```\n
\n停止服务\n\n
\n```\n`bash mtproxy.sh stop
`\n```\n
\n重启服务\n\n
\n```\n`bash mtproxy.sh restart
`\n```\n
\n卸载安装
因为是绿色版卸载极其简单，直接删除所在目录即可。\n\n
\n```\n`rm -rf /home/mtproxy
`\n```\n

---

\nps:安装完成后还要开启相对应的端口，本实例需开启443端口，代码如下：\n\n
\n```\n`firewall-cmd --zone=public --add-port=443/tcp --permanent
firewall-cmd --reload
`\n```\n
\n关闭端口命令如下：\n\n
\n```\n`firewall-cmd --zone=public --remove-port=443/tcp --permanent
firewall-cmd --reload
`\n```\n
\n卸载
因为是绿色版卸载极其简单，直接删除程序目录即可；\n\n
\n```\n`rm -rf /home/mtproxy
`\n```\n
\n开机启动
编辑/etc/rc.local开机自启服务文件，将如下代码添加到开机自启脚本中；\n\n
\n编辑自启文件
vi /etc/rc.local\n\n
\n添加如下代码\n\n
\n```\n`bash /home/mtproxy/mtproxy.sh start > /dev/null 2>&1 &
`\n```\n
\n添加rc.local执行权限\n\n
\n```\n`chmod +x /etc/rc.d/rc.local
`\n```\n
\n重启系统查看是否成功启动\n\n
\n```\n`reboot
`\n```\n
