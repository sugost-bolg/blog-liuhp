\n在 root 下添加为开机自启（使用 systemd，服务以 root 身份运行）。将下面内容写入 /etc/systemd/system/okx_multi_account.service 并启用：\n\n\n1. 创建 unit 文件：\n\nCode\n\n\n```\n`sudo tee /etc/systemd/system/okx_multi_account.service > /dev/null <<'EOF'
Description=okx_multi_account Python service (run as root)
WorkingDirectory=/home/auto_trade
ExecStart=/usr/bin/python3 /home/auto_trade/okx_multi_account.py
sudo systemctl enable --now okx_multi_account.service`\n```\n\n1. 管理与查看：\n\nCode\n\n\n```\n`sudo systemctl status okx_multi_account.service
sudo journalctl -u okx_multi_account.service -f
sudo systemctl stop okx_multi_account.service
sudo systemctl disable okx_multi_account.service
`\n```\n\n注意：\n\n- 确认 /home/auto_trade/okx_multi_account.py 可被 root 读取并可运行（通常 root 有权限）。\n- 若脚本需虚拟环境或特定 PATH，把 ExecStart 改为该虚拟环境的 python 路径。\n\n\n\n
