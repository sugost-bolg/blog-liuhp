                                                                                                                                                        </p><p style="">                                                                                                                                                                                   </p><p style="">   # OpenClaw 一键备份还原系统                                                                                                                                                     </p><p style="">                                                                                                                                                                                   </p><p style="">   为 OpenClaw 设计的一套极简备份方案，基于 Git 单仓标签管理，支持文件变更自动备份和一键重装恢复。                                                                                 </p><p style="">                                                                                                                                                                                   </p><p style="">   ## 设计原则                                                                                                                                                                     </p><p style="">                                                                                                                                                                                   </p><p style="">   - <strong>单仓多分支</strong>：一个 Git 仓库，用标签管理版本                                                                                                                                 </p><p style="">   - <strong>自动轮转</strong>：保留最近 3 次 + 当前，老的自动清理                                                                                                                              </p><p style="">   - <strong>零依赖自举</strong>：重装后一行命令就能开始恢复                                                                                                                                    </p><p style="">   - <strong>双轨推送</strong>：SSH 用于日常备份，Git Daemon 用于紧急恢复                                                                                                                       </p><p style="">                                                                                                                                                                                   </p><p style="">   ## 备份策略：标签轮转                                                                                                                                                           </p><p style="">                                                                                                                                                                                   </p><p style="">   | 标签名 | 含义 | 生命周期 |                                                                                                                                                    </p><p style="">   |--------|------|----------|                                                                                                                                                    </p><p style="">   | <code>current</code> | 当前工作副本 | 持续更新 |                                                                                                                                         </p><p style="">   | <code>backup-1</code> | 最新备份 | 保留 |                                                                                                                                                </p><p style="">   | <code>backup-2</code> | 上次备份 | 保留 |                                                                                                                                                </p><p style="">   | <code>backup-3</code> | 上上次备份 | 保留 |                                                                                                                                              </p><p style="">                                                                                                                                                                                   </p><p style="">   ## 命令                                                                                                                                                                         </p><p style="">                                                                                                                                                                                   </p><p style="">   ```bash                                                                                                                                                                         </p><p style="">   backup now          # 立即备份                                                                                                                                                  </p><p style="">   backup list         # 列出可用备份                                                                                                                                              </p><p style="">   backup restore 2    # 恢复到指定版本                                                                                                                                            </p><p style="">   backup auto         # 启动自动监听                                                                                                                                              </p><p style="">   backup stop         # 停止自动监听                                                                                                                                              </p><p style=""> ```                                                                                                                                                                               </p><p style="">                                                                                                                                                                                   </p><p style=""> 重装恢复                                                                                                                                                                          </p><p style="">                                                                                                                                                                                   </p><p style=""> ```bash                                                                                                                                                                           </p><p style="">   curl git://10.10.10.230/openclaw-back up.git/restore.sh | bash                                                                                                                  </p><p style=""> ```                                                                                                                                                                               </p><p style="">                                                                                                                                                                                   </p><p style=""> 开机自启动                                                                                                                                                                        </p><p style="">                                                                                                                                                                                   </p><p style=""> 通过 macOS LaunchAgent 实现，监听文件变更，5秒后自动备份。
